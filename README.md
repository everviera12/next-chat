## Run Locally

Clone the project

```bash
  git clone https://github.com/everviera12/next-chat
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  pnpm install
```

Start the server

```bash
  pnpm run dev
```
